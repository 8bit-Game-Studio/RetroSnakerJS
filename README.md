# RetroSnakerJS


贪吃蛇 JS 版


## 基本操作

方向键： ⬆️ ⬇️ ⬅️ ➡️ 控制移动；

空格键： 暂停／继续；

R：重置游戏。

## Basic

Direction key: ⬆️ ⬇️ ⬅️ ➡️ control the direction of the snaker;

Space key: Pause/Continue;

R: Reset game.
